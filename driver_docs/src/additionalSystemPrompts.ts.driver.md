# Purpose
This JavaScript file defines a constant array named `additionalSystemPrompts`, which contains a series of string prompts designed for an AI-powered stock screener application. The prompts provide detailed instructions and schemas for generating BigQuery SQL queries based on user input, focusing on financial data such as stock earnings, price data, and industry classifications. The code serves a narrow functionality, specifically tailored to guide an AI in translating natural language queries into SQL queries for a stock screening platform. The prompts include constraints, objectives, and examples to ensure the AI generates accurate and efficient queries while adhering to specific guidelines, such as avoiding SQL injection attacks and ensuring the inclusion of necessary fields and calculations. The file is essentially a configuration or setup script that provides the AI with the necessary context and rules to perform its task effectively.
