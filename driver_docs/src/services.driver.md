
## Files
- **[AnthropicServiceClient.ts](services/AnthropicServiceClient.ts.driver.md)**: The `AnthropicServiceClient.ts` file implements a client for interacting with the Anthropic AI service, handling chat message requests and responses, and includes functionality for retrying requests and logging chat interactions.
- **[bigQueryClient.ts](services/bigQueryClient.ts.driver.md)**: The `bigQueryClient.ts` file in the `Promptimizer` codebase defines a `BigQueryDataManager` class that manages Google BigQuery operations, including setting up credentials and executing SQL queries.
- **[chatController.ts](services/chatController.ts.driver.md)**: The `chatController.ts` file in the `Promptimizer` codebase implements a chat controller that handles AI-generated chat messages, ensuring they conform to a specified JSON schema when required.
- **[db.ts](services/db.ts.driver.md)**: The `db.ts` file in the `Promptimizer` codebase defines a class for managing connections to MongoDB databases, supporting local, cloud, and in-memory test environments.
- **[GenAIFactory.ts](services/GenAIFactory.ts.driver.md)**: The `GenAIFactory.ts` file defines a factory class for creating instances of generative AI service clients based on the specified model type, including OpenAI, Anthropic, and open-source models.
- **[GenerativeAIServiceClient.ts](services/GenerativeAIServiceClient.ts.driver.md)**: The `GenerativeAIServiceClient.ts` file defines an abstract class for interacting with a generative AI service, including methods for transforming chat messages and handling message roles.
- **[OpenAIServiceClient.ts](services/OpenAIServiceClient.ts.driver.md)**: The `OpenAIServiceClient.ts` file implements a client for interacting with OpenAI's API, providing functionalities for generating embeddings, analyzing images, and sending chat messages while handling API key management and error logging.
- **[OpenSourceServiceClient.ts](services/OpenSourceServiceClient.ts.driver.md)**: The `OpenSourceServiceClient.ts` file defines a client for interacting with open-source AI services, including functionalities for image analysis, sending chat requests, and generating text embeddings.
