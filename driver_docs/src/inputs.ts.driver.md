# Purpose
This JavaScript file defines and exports a constant array named `inputs`, which contains a collection of objects representing financial queries categorized by difficulty levels: easy, medium, and hard. Each object in the array includes a `text` property, which is a string describing a specific financial question, and a `foldername` property, which provides a corresponding identifier for organizational purposes. The code serves as a configuration or data file, providing a structured list of financial questions that can be used in applications such as financial data analysis tools, educational platforms, or quiz applications. The functionality is narrow, focusing specifically on organizing and categorizing financial queries for further processing or display.
# Global Variables

### inputs
- **Type**: `array`
- **Description**: The `inputs` variable is an array of objects, each representing a financial question related to various companies and sectors. Each object contains a `text` property with the question and a `foldername` property that categorizes the question by difficulty and topic.
- **Use**: This variable is used to store and organize a list of financial questions for further processing or querying.
