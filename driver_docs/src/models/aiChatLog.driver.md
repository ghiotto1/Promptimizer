
## Files
- **[anthropic.ts](aiChatLog/anthropic.ts.driver.md)**: The `anthropic.ts` file defines a Mongoose model and a logging mechanism for storing chat interactions with the Anthropic AI, including request and response details, in the `Promptimizer` codebase.
- **[openai.ts](aiChatLog/openai.ts.driver.md)**: The `openai.ts` file defines interfaces and a Mongoose model for logging OpenAI chat, image, and embedding requests and responses, including error handling and saving logs to a database.
- **[opensource.ts](aiChatLog/opensource.ts.driver.md)**: The `opensource.ts` file in the `Promptimizer` codebase defines a Mongoose model and methods for logging different types of AI chat interactions, including image requests, embeddings, and chat messages, along with any associated errors.
- **[shared.ts](aiChatLog/shared.ts.driver.md)**: The `shared.ts` file defines the `SendChatMessageRequest` interface for sending chat messages and the `AiChatTypeEnum` enumeration for specifying AI chat types in the Promptimizer codebase.
