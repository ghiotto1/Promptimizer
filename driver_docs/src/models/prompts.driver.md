
## Files
- **[abstract.ts](prompts/abstract.ts.driver.md)**: The `abstract.ts` file in the `Promptimizer` codebase defines the `AbstractPrompt` class, which models a prompt with attributes such as name, description, examples, and model type, and provides methods for saving, deleting, and retrieving prompts from a MongoDB database using Mongoose.
- **[defaults.ts](prompts/defaults.ts.driver.md)**: The `defaults.ts` file in the `Promptimizer` codebase defines default configurations for AI prompts related to stock screening and evaluation, including detailed instructions and scoring criteria for evaluating AI-generated outputs against ground truth data.
- **[example.ts](prompts/example.ts.driver.md)**: The `example.ts` file defines a `PromptExample` class that represents a conversation consisting of an array of `ChatMessage` objects, with optional properties for an ID and name.
