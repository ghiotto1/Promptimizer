
## Files
- **[input.txt](easy_microsoft_free_cash_flow/input.txt.driver.md)**: The `input.txt` file contains a prompt asking for the free cash flow of Microsoft for the latest fiscal year.
- **[output.txt](easy_microsoft_free_cash_flow/output.txt.driver.md)**: The `output.txt` file contains instructions and an SQL query to calculate the total free cash flow for Microsoft for the latest fiscal year by summing the free cash flow values from the `earnings` table.
