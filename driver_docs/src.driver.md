## Folders
- **[groundTruths](src/groundTruths.driver.md)**: The `groundTruths` folder in the `Promptimizer` codebase contains a comprehensive collection of subfolders, each dedicated to querying and explaining financial data retrieval for various companies and sectors using SQL, categorized by difficulty levels such as easy, medium, and hard.
- **[models](src/models.driver.md)**: The `models` folder in the `Promptimizer` codebase contains TypeScript files and folders that define Mongoose models and interfaces for managing AI chat logs, context objects, chat messages, and AI prompts, along with shared utilities for handling object IDs.
- **[services](src/services.driver.md)**: The `services` folder in the `Promptimizer` codebase contains various TypeScript files that implement clients and controllers for interacting with different AI services, managing database connections, and handling data operations, including support for Anthropic, OpenAI, and open-source models.
- **[utils](src/utils.driver.md)**: The `utils` folder in the `Promptimizer` codebase contains utility functions for JSON manipulation, array operations, and promise error handling, as defined in the `index.ts` file.

## Files
- **[additionalSystemPrompts.ts](src/additionalSystemPrompts.ts.driver.md)**: The `additionalSystemPrompts.ts` file in the `Promptimizer` codebase contains a collection of system prompts designed to guide an AI-powered stock screener in generating BigQuery SQL queries based on user input, with detailed instructions and constraints for handling financial data from specified datasets.
- **[inputs.ts](src/inputs.ts.driver.md)**: The `inputs.ts` file in the `Promptimizer` codebase defines a list of financial questions categorized by difficulty, each associated with a specific folder name for organizational purposes.
- **[main.ts](src/main.ts.driver.md)**: The `main.ts` file in the `Promptimizer` codebase orchestrates the process of optimizing prompts through a genetic algorithm, including data preparation, prompt creation, population management, and generation of summaries.
- **[populateGroundTruth.ts](src/populateGroundTruth.ts.driver.md)**: The `populateGroundTruth.ts` file in the `Promptimizer` codebase implements a script to generate and validate SQL queries against a database, using user input and manual review to ensure accuracy, and stores the results in a specified directory.
- **[population.ts](src/population.ts.driver.md)**: The `population.ts` file in the `Promptimizer` codebase defines classes and methods for managing a population of prompt-based individuals, including their creation, testing, crossover, mutation, and evaluation within a genetic algorithm framework.
